Cookie Killer – Privacy Policy

Cookie Killer does not collect, transmit, or share any personal data.

The only data stored by the extension is a whitelist of domain names that you explicitly add. This list is saved locally in your browser using the browser's built-in storage API (chrome.storage.sync). If you are signed into Chrome, this list may sync across your own devices via your Google account — no third party ever receives it.

The extension does not make any network requests of its own. It does not track browsing history, page content, or any user activity.
